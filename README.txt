Simon Game

Developed simon game where the site presses a button and after each press the user has to remember the pattern from the start and press the whole pattern to process forward to the next level.

Motivation

Developed it while completing the course "The Complete 2022 Web Development Bootcamp". Aim wwas to learn creating animation using different methods in Javascript and event handlers and functions.

Tech/Framework used

HTML, CSS, and JavaScript.

Tests

Press a key to start the game.
